BEM-express
===========

BEM driver for expressjs

### Install
 1. `git clone git://github.com/Navotno/bem-express.git`
 2. `cd bem-express`
 3. `npm install && ./node_modules/.bin/bem make`

### Usage
#### Development environment
 1. `node app.js`
 2. `http://127.0.0.1:3000/`

#### Testing and production environment
 1. `make`
 2. `http://127.0.0.1:3000/`
